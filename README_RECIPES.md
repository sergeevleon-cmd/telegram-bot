# 🍽️ Telegram Бот для Рецептов

Бот для поиска рецептов через TheMealDB API с возможностью сохранения избранных рецептов.

## 📁 Структура проекта

```
Telegram/
├── bot.py              # Основной файл бота
├── database.py         # Работа с SQLite базой данных
├── recipes_api.py      # API функции для TheMealDB
├── keyboards.py        # Клавиатуры и кнопки
├── .env                # Токен бота и настройки
├── requirements.txt    # Зависимости Python
├── recipes.db          # База данных (создается автоматически)
└── logs/               # Логи бота
    └── bot.log
```

## 🚀 Запуск

```bash
python bot.py
```

- ⬅️ Назад в меню

## 🎯 Функционал

### Поиск рецептов
1. **Случайный рецепт** - получить случайное блюдо
2. **По категориям** - выбрать из 12+ категорий (Chicken, Seafood, Vegetarian и т.д.)
3. **По кухням мира** - выбрать из 18+ кухонь (Italian, Chinese, Mexican и т.д.)
4. **По названию** - поиск по введенному названию

### Избранное
- Добавление рецептов в избранное (кнопка ⭐)
- Просмотр всех сохраненных рецептов
- Удаление рецептов из избранного
- Очистка всего избранного

## 🗄️ База данных

### Таблица `favorites`
| Поле       | Тип       | Описание                    |
|------------|-----------|----------------------------|
| id         | INTEGER   | Первичный ключ             |
| user_id    | INTEGER   | ID пользователя Telegram   |
| meal_id    | TEXT      | ID рецепта из TheMealDB    |
| meal_name  | TEXT      | Название рецепта           |
| meal_thumb | TEXT      | URL изображения            |
| added_at   | TIMESTAMP | Дата добавления            |

## 🔧 API TheMealDB

Используемые эндпоинты:
- `random.php` - случайный рецепт
- `search.php?s={name}` - поиск по названию
- `categories.php` - список категорий
- `filter.php?c={category}` - рецепты по категории
- `list.php?a=list` - список кухонь мира
- `filter.php?a={area}` - рецепты по кухне
- `lookup.php?i={id}` - детали рецепта по ID

## 📦 Зависимости

- pyTelegramBotAPI - работа с Telegram Bot API
- requests - HTTP запросы к TheMealDB API
- python-dotenv - загрузка переменных окружения
- sqlite3 - встроенная БД (не требует установки)
