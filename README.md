# Telegram bot template (aiogram)

## Что делает бот
- `/start` - приветствует
- любое сообщение, где встречается слово `Привет` (регистр не важен) - снова приветствует
- команды для проверки:
  - `/help` - список команд
  - `/ping` - отвечает `pong`
  - `/test` - отвечает `Команды работают`

## Требования
- Python 3.10+

## Установка и запуск
1. Установите зависимости:
   - `python -m venv venv`
   - активируйте venv:
     - PowerShell: `.\venv\Scripts\Activate.ps1`
   - `pip install -r requirements.txt`
2. Создайте файл `.env` на основе `.env.example` и укажите `BOT_TOKEN`.
3. Запустите:
   - `python bot.py`

## Тестирование
- Откройте бота в Telegram и отправьте `/start`
- Отправьте сообщение `Привет` (или `привет`, `ПРИВЕТ`) и убедитесь, что бот ответил
- Попробуйте `/help`, `/ping`, `/test`
